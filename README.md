## 📖 Sobre o Projeto

Este repositório contém o código-fonte do **novo site institucional da Areatec**, líder brasileira em tecnologia para fiscalização de trânsito e estacionamento rotativo. 

O projeto foi desenvolvido utilizando o conceito visual *"Clean / Notion-Stripe Hybrid"*, com amplo uso de espaço em branco, tipografia moderna (Manrope + Source Sans 3) e o característico vermelho Areatec para destaques. O objetivo é substituir o atual site `areatec.com.br` por uma plataforma mais rápida, responsiva e com foco em conversão e apresentação clara dos produtos.

## 🚀 Nossos Produtos

O novo site apresenta o portfólio completo de soluções da Areatec:

1. **Veículo OCR:** Fiscalização móvel inteligente
2. **Cortex Areatec:** Inteligência Artificial aplicada ao trânsito
3. **Olho Vivo:** Monitoramento avançado por câmeras
4. **Digipare:** Solução líder em estacionamento rotativo
5. **Talonário Eletrônico:** Agilidade na fiscalização
6. **SyncRealTime:** Sincronização de dados em tempo real
7. **GeoTrust (OSNMA/Galileo):** Autenticação de posicionamento global
8. **AreaChain:** Blockchain privada para auditoria de infrações

## ⚙️ Como Rodar Localmente

### Pré-requisitos
- Node.js (v18 ou superior)
- pnpm (v10.x recomendado)

### Instalação e Execução

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/fabiobatistella/novo-site-areatec.git
   cd novo-site-areatec
   ```

2. **Instale as dependências:**
   ```bash
   pnpm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   pnpm run dev
   ```

4. Acesse `http://localhost:3000` no seu navegador.

### Outros Comandos Úteis
- `pnpm run build`: Gera a versão de produção
- `pnpm run preview`: Visualiza a versão de produção localmente
- `pnpm run format`: Formata o código com Prettier
- `pnpm run check`: Verifica erros de TypeScript

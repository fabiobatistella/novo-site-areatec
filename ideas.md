## Ideia 1: "Cinematic Light Precision"
**Movimento de Design:** Cinematografia Corporativa Minimalista — inspirado em filmes de ficção científica como Arrival e Ex Machina, mas em modo light.

**Princípios Fundamentais:**
1. Clareza através de hierarquia visual rigorosa
2. Movimento como narrativa — cada animação conta uma história
3. Espaço negativo como elemento de design ativo
4. Precisão tipográfica com contraste dramático

**Filosofia de Cor:**
- Fundo predominantemente branco (#FAFBFC) com gradientes sutis de cinza
- Vermelho Areatec (#DC2626) como cor primária de destaque em ícones
- Tiffany (#0ABAB5) como cor de energia/vida nos elementos pulsantes
- Cinza escuro (#1A1A2E) para textos principais
- Gradientes sutis de branco para cinza claro nas seções alternadas

**Paradigma de Layout:** Seções full-width com conteúdo centralizado em container max-w-7xl, alternando entre layouts assimétricos (texto+imagem) e grids de cards. Uso de parallax sutil e scroll-triggered animations.

**Elementos Assinatura:**
1. Ícones tiffany pulsantes com glow effect espalhados por todo o site
2. Linhas de conexão animadas entre seções (como circuitos)
3. Badges com backdrop-blur e borda sutil

**Filosofia de Interação:** Cada hover revela uma camada adicional de informação. Transições suaves de 300-500ms. Elementos entram com fade-in + slide-up ao scroll.

**Animação:** Scroll-triggered fade-in com stagger para cards. Pulse animation contínua nos ícones tiffany. Counter animation nos números. Parallax sutil no hero.

**Sistema Tipográfico:** Sora para títulos (bold, impactante), DM Sans para corpo (legível, moderno). Hierarquia: 4xl-5xl para h1, 3xl para h2, xl para h3, base para corpo.

## Ideia 2: "Urban Tech Editorial"
**Movimento de Design:** Editorial Digital — inspirado em revistas de tecnologia premium como Wired e MIT Technology Review.

**Princípios Fundamentais:**
1. Grid editorial com colunas assimétricas
2. Tipografia como elemento gráfico principal
3. Imagens cinematográficas como pontos focais
4. Micro-interações que recompensam a exploração

**Filosofia de Cor:**
- Base off-white (#F8F9FA) com seções alternando para cinza muito claro
- Vermelho Areatec como accent em ícones e CTAs
- Tiffany como cor de "vida digital" nos pulsos
- Preto profundo (#0F172A) para títulos
- Gradientes de vidro (glassmorphism sutil) nos cards

**Paradigma de Layout:** Grid editorial com sidebar informativa. Seções com larguras variáveis. Hero com split-screen diagonal. Cards com aspect ratios variados criando ritmo visual.

**Elementos Assinatura:**
1. Números grandes como elementos decorativos
2. Linhas horizontais finas separando conteúdo
3. Badges com ícones pulsantes em tiffany

**Filosofia de Interação:** Hover com scale sutil e shadow elevation. Cards que revelam informação adicional. Scroll progress indicator.

**Animação:** Entrada editorial com texto aparecendo palavra por palavra. Cards com stagger delay. Imagens com zoom sutil no scroll.

**Sistema Tipográfico:** Source Sans 3 para títulos, DM Sans para corpo. IBM Plex Mono para dados técnicos e badges.

## Ideia 3: "Neural Network Aesthetic"
**Movimento de Design:** Tech-Organic — inspirado em interfaces de IA e visualizações de redes neurais, mas clean e light.

**Princípios Fundamentais:**
1. Conexões visuais entre elementos (como sinapses)
2. Dados como decoração — números e métricas são visuais
3. Profundidade através de camadas de transparência
4. Movimento orgânico em elementos técnicos

**Filosofia de Cor:**
- Branco puro com micro-texturas de grid
- Tiffany (#0ABAB5) como "energia neural" pulsante
- Vermelho Areatec para ações e ícones
- Gradientes de tiffany-para-transparente nos glows
- Cinza neutro (#374151) para texto

**Paradigma de Layout:** Layout fluido com elementos que parecem "flutuar". Cards com bordas luminosas. Seções conectadas por linhas animadas. Grid orgânico que se adapta ao conteúdo.

**Elementos Assinatura:**
1. Dots pulsantes em tiffany conectados por linhas finas
2. Glow effects nos hover states
3. Background com grid pattern sutil

**Filosofia de Interação:** Elementos reagem ao proximity do mouse. Hover cria "ondas" de energia. Cliques geram feedback visual imediato.

**Animação:** Pulse contínuo nos nós da rede. Linhas que se desenham ao scroll. Partículas flutuantes no background.

**Sistema Tipográfico:** Sora para impacto nos títulos, DM Sans para legibilidade no corpo. Monospace para dados técnicos.

## Decisão: Ideia 1 — "Cinematic Light Precision"

Escolho a Ideia 1 por ser a mais alinhada com o site original (cinematográfico light) e por oferecer a melhor base para os ícones tiffany pulsantes solicitados.

## [1.0.0] - 2024-05-08

### Adicionado
- Estrutura inicial do projeto com React 19, TypeScript e Vite.
- Configuração do Tailwind CSS v4 e Framer Motion.
- Componentes base da UI utilizando Radix UI.
- Layout principal e navegação.
- Seções da página inicial (Hero, Soluções, Cortex, Diferenciais, etc).
- Integração de assets estáticos (imagens, vídeos).
- Suporte inicial a tema claro/escuro.

### Modificado
- Refinamento do design visual seguindo o conceito "Clean / Notion-Stripe Hybrid".
- Correções de responsividade nos componentes.
- Ajuste no alinhamento das badges da seção Cortex.

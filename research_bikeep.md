# Bikeep - Resumo para Seção de Soluções Areatec

- Estações inteligentes de estacionamento para bicicletas e patinetes
- Stations + App + Platform (ecossistema completo)
- +10.000 dispositivos instalados em +30 países, 13+ anos
- Produtos: Secure Bike Parking, Secure Scooter Parking & Charging, E-bike Charging, Battery Powered Station
- App para lock/unlock via QR code
- Management Console (dashboard cloud)
- Clientes: aeroportos (Reagan National), universidades (UNSW Sydney), cidades (Vancouver TransLink)
- Areatec é representante no Brasil - parceria estratégica
- Posicionamento: infraestrutura de micromobilidade para cidades inteligentes
